# RSSHub Docs

## 为什么

官方文档太卡，打算建立一个项目用来重构

## 内容

只会收录一般大众使用的服务，不常用的服务将不会收录到文档以保证速度和优化。

## 使用

```
git clone https://github.com/JimmyRice/rsshubdocs.git rssdocs

cd rssdocs

sudo npm install

```

### 文档目录：
./docs/content/about:关于页面
./docs/content/install:部署页面
./docs/content/api:指南页面
